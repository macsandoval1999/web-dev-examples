⚙️ Using sessionStorage

This JavaScript code is designed to capture a user's name from an input field and store it temporarily using the browser's sessionStorage API. It then uses the stored name to display a personalized greeting upon subsequent page visits within the same browsing session.

  1. Enter your name
  2. Click "Store Name"
  3. Refresh the page
    A welcome message should be visible
  4. Close the tab and reopen the URL in a new tab
    There will be no welcome message because sessionStorage is cleared when the tab is closed.
  5. Copy this CodePen URL from the URL address bar
  6. Open a new tab and paste the URL in the address bar and press enter There will be no welcome message.
