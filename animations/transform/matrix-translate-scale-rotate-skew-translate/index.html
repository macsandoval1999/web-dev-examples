transform: matrix() does a combination of translate, scale, rotate, and skew transformations in one function.


transform: translate() moves an element from its current position.

transform: translateX() moves an element along the X axis.

transform: translateY() moves an element along the Y axis.

transform: translateZ() moves an element along the Z axis.

transform: translate3d() moves an element in 3D space along the X, Y, and Z axes.


transform: scale() resizes an element.

transform: scaleX() resizes an element along the X axis.

transform: scaleY() resizes an element along the Y axis.

transform: scaleZ() resizes an element along the Z axis.

transform: scale3d() resizes an element in 3D space along the X, Y, and Z axes.


transform: rotate() rotates an element around a fixed point.

transform: rotateX() rotates an element around the X axis.

transform: rotateY() rotates an element around the Y axis.

transform: rotateZ() rotates an element around the Z axis.

transform: rotate3d() rotates an element in 3D space around the X, Y, and Z axes.


transform: skew() skews an element along the X and Y axes.

transform: skewX() skews an element along the X axis.

transform: skewY() skews an element along the Y axis.
